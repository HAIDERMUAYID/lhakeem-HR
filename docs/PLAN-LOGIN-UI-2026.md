# خطة واجهة تسجيل الدخول والواجهة العامة 2026

## الأهداف
- تصميم **لاندسكيب** يستغل عرض الشاشة.
- مظهر **احترافي "مثل الفدير"**: أنيق، هرمية واضحة، ألوان مؤسسية.
- **شرح مزايا النظام** على صفحة الدخول.
- **جميع الحقوق محفوظة لمستشفى الحكيم العام © 2026** في التذييل.

---

## 1. صفحة تسجيل الدخول (لاندسكيب + مزايا + حقوق)

| العنصر | الوصف |
|--------|--------|
| **التخطيط** | تقسيم أفقي: قسم يسار (تعريف + مزايا) وقسم يمين (نموذج الدخول). على الشاشات الصغيرة يبقى النموذج فقط مع مزايا مضغوطة أسفل. |
| **القسم الأيسر** | شعار، اسم النظام، جملة ترحيب، وقائمة مزايا مختصرة (إجازات، موظفين، أقسام، بصمة، تقارير، صلاحيات). |
| **القسم الأيمن** | بطاقة تسجيل الدخول الحالية مع تحسين ظل وحواف. |
| **التذييل** | سطر واحد: «جميع الحقوق محفوظة لمستشفى الحكيم العام © 2026» مع رمز ® إن رغبت. |

## 2. مزايا النظام (للعرض على الدخول)

- **الإجازات وتقويم الإجازات** — طلبات إجازة، اعتماد، أنواع إجازات.
- **الموظفين والأقسام** — إدارة بيانات الموظفين والأقسام والاستيراد.
- **البصمة والغيابات** — أجهزة البصمة، تقويم وحدة البصمة، تسجيل الغيابات.
- **جداول الدوام والعطل** — إدارة الدوام والعطل الرسمية.
- **التقارير وسجل التدقيق** — تقارير وإحصائيات وتتبع العمليات.
- **المستخدمون والصلاحيات** — إدارة المستخدمين والصلاحيات حسب الدور.

## 3. لوحة التحكم (الداشبورد)

- إضافة **تذييل ثابت** في الـ layout يحتوي على: «جميع الحقوق محفوظة لمستشفى الحكيم العام © 2026» ليكون ظاهراً في كل صفحات لوحة التحكم.

## 4. معايير التصميم "مثل الفدير"

- ألوان هادئة ومؤسسية (أزرق، رمادي، أبيض).
- مسافات كافية وعدم ازدحام.
- خط عربي واضح وهرمية قراءة واحدة.
- ظلال خفيفة، حواف مستديرة، لا حركات مزعجة.
